Import / Export Settings

Transfer your SpotOn settings between different devices and browsers with ease. Follow these steps:

Exporting Your Settings

  1. Click the "Export Options" button to generate a JSON file containing your current settings such as toggles and custom colors.
  2. A file will be downloaded to your device. Keep this file safe as it contains your SpotOn settings.

Importing Your Settings

  1. On the device or browser where you want to apply the settings, open the SpotOn settings page.
  2. Click the "Import Options" button to reveal a file input field.
  3. Use the file input to select the previously downloaded JSON settings file.
  4. Once the file is selected, your settings will be automatically applied to SpotOn on this new device or browser.

If it doesn't automatically reload, then go out and back to the homepage and press "Save Options." This will ensure that everything is then saved and loaded in!
