SpotOn Extension Help

Welcome to the help page for the SpotOn. Here, you can learn how SpotOn works, test some customization, learn various settings and get help with various settings and how they work and function!

General Options

There are two parts of SpotOn. Toggle options, like you see in the button above, and Custom options, where you can define your OWN values for certain settings! We will explore the associated options in-depth below.

Navigating to the Settings page?

To navigate to the settings page, open the extension and click on "settings". Once the settings page is open, you'll find several sections such as "Main Options", "Interface Options", "Theming Options", "Removal Options", and "Custom Options". At the bottom of the menu bar, there's a "Save Options" button. To save any option, whether it's a custom or toggle option, simply press the "Save Options" button and then reload Spotify.

If you cannot find something, try the above search button! Do note, you'll need to go into each category separately to find the value, for an example, try search "Rainbow".

Or, you can simply follow this step by step photos First open the popup!
We then want to press settings, after pressing settings, you'll be launched into the main tab.

You can then either find the option or use the search bar, for this example, we'll use "rainbow controls", we can find this using the search bar!

TIP: If you cannot find the option you are looking for, look in every tab, if it's not there, submit it as feedback!

After you have got your options sorted, press "Save Options" and you're done!

Use these keyboard shortcuts to navigate the extension:

Glossary

Toggle Options
Options that enable or disable specific features.
RGBA
A color model that includes red, green, blue, and alpha (transparency) values.
RGB
A color model that includes red, green, and blue values.
Hsl
The HSL model (which HSL stands for Hue, Saturation, and Lightness) is widely considered to be more intuitive than the RGB model. This is because, the HSL model allows you to select a base hue, and then adjust its saturation and lightness as desired.
Hex
A color model that allowing designers and developers to easily communicate about color schemes. With 16 × 16 (256) values for each color component, there are then 256 × 256 × 256 possible combinations. In hex, they range from #000000 (pure black) to #FFFFFF (pure white).
NPB
Spotifys Now Playing Bar (where you're controls are, play pause, skip etc)
NB
Spotifys Nav (or navigation) bar (right hand side bar)
NPV
Spotifys new Now Playing View which shows the song, and info about it on the left side, button can be found on the NPB

Frequently Asked Questions

How do I disable SpotOn?

Go to General Options and check the "Enable SpotOn" checkbox, then save.

How do I change themes and not use SpotOn's default theme?

Go to the main popup under "SpotOn Theme" press the dropdown, select your theme, then press "save" at the bottom

How can I report an issue?

You can report issues via the extension popup by clicking "Report Issue" or on our discord.
